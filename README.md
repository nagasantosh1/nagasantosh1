# 💫 About Me:

My work centers around Agentic AI, multi-agent architectures, and retrieval-augmented systems, where I design solutions that go beyond prototypes and deliver measurable business impact.<br><br>At AIONOS (client: IndiGo), I developed an AI-powered invoice generation system that automates national and international billing workflows using multi-agent orchestration, RAG, and dynamic validation.
